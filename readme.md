# How to Use

You should place pc_nsf_hifigan_44.1k_hop512_128bin_2025.02.onnx in the root directory.

Setup Venv
  - setup-venv.ps1

Launch the script with:
  - launch.ps1

Package using (After setting up and successful launching):
  - package.ps1
